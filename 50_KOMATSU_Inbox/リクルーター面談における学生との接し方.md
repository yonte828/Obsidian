# リクルーター面談における学生との接し方

リクルーター面談は、採用選考の一環と見なされることが多いため、カジュアルな雰囲気の中でもプロ意識を持った対応が重要です。

### 上手な接し方
1.  **カジュアルさの中にもプロ意識を**: カフェなどで行われる場合でも、社会人としての基本的なマナー（服装、言葉遣い、態度）を保ち、企業を代表する立場であることを意識する。
2.  **傾聴と共感**: 学生の話に真摯に耳を傾け、彼らの経験や価値観を尊重する姿勢を見せることで、学生に安心感を与える。
3.  **事前の準備**: 学生のプロフィールや、自社の情報、特に学生が興味を持ちそうな点について準備しておく。
4.  **相互理解の促進**: 面談の目的は企業と学生の双方にとってのミスマッチを防ぐこと。学生が企業を深く理解できるよう、リアルな情報を提供し、疑問や不安を解消する手助けをする。
5.  **選考への影響を意識**: 面談は後の選考に影響を与える可能性があるため、学生のコミュニケーション能力や企業への志望度、ビジネスマナーなどが評価されていることを意識して接する。
6.  **過度な馴れ馴れしさを避ける**: たとえOB・OGであっても、親密すぎる接し方は避け、節度を保つことが望ましい。

### 学生への質問例
-   **自己紹介・自己PR**: 学生の基本的な情報や強みを知る。
-   **志望動機**: 自社への熱意や企業理解度を確認する。
-   **長所・短所**: 自己認識の深さや成長意欲を見る。
-   **学生時代に力を入れたこと（ガクチカ）**: 経験から何を学び、どのように行動したかを通じて、潜在能力や価値観を探る。
-   **企業選びの軸**: 学生がどのような基準で企業を選んでいるかを知り、自社とのマッチング度を測る。
-   **入社後にやりたいこと・キャリアプラン**: 学生の将来の展望や、自社でどのように貢献したいと考えているかを確認する。

### 学生からの逆質問への対応
学生からの逆質問は、企業への関心度や理解度を示す重要な機会。リクルーターは、以下の質問に具体的に答えられるように準備しておく。
-   企業文化や社風
-   具体的な業務内容や一日のスケジュール
-   キャリアパスや成長機会
-   仕事のやりがいや難しさ
-   企業が求める人物像
-   業界の動向や競合との違い
-   研修制度やサポート体制
-   ワークライフバランス
