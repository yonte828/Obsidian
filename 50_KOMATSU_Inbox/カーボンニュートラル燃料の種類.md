# カーボンニュートラル燃料の種類

カーボンニュートラル燃料とは、その製造から利用までのライフサイクル全体で、大気中の二酸化炭素（CO2）を増加させないとされる燃料のことです。地球温暖化対策の一環として、運輸部門や産業部門での脱炭素化に向けた重要な選択肢とされています。

HVO（水素化処理植物油）もその一種ですが、他にも多様な種類が存在します。

### 1. バイオ燃料 (HVO以外)
生物資源（バイオマス）を原料とする燃料。原料となる植物が光合成でCO2を吸収するため、燃焼時にCO2を排出しても実質的にプラスマイナスゼロになると考えられています。

- **バイオエタノール**:
    - **原料**: サトウキビ、トウモロコシ、テンサイなど（糖質・デンプン質系）、または稲わら、木材チップなど（セルロース系）。
    - **用途**: 主にガソリン代替燃料。
- **バイオディーゼル (FAME: 脂肪酸メチルエステル)**:
    - **原料**: 廃食用油、菜種油、パーム油など。
    - **用途**: 軽油代替燃料。HVOとは製造方法が異なります。
- **バイオガス**:
    - **原料**: 家畜排せつ物、生ごみ、下水汚泥など。
    - **用途**: メタンを主成分とし、発電や都市ガスの原料として利用。

### 2. 合成燃料 (e-fuel)
再生可能エネルギーなどから製造した水素（H2）と、工場などから回収した二酸化炭素（CO2）を原料として合成される燃料です。「電気（electro-）」を元に製造されることから「e-fuel」とも呼ばれます。

- **製造プロセス (Power-to-Liquid)**:
    1. 再生可能エネルギー（太陽光、風力など）で水を電気分解し、水素を製造。
    2. 発電所や工場から排出されるCO2を回収。
    3. 水素とCO2を合成して、原油に近い液体燃料（合成原油）を生成。
    4. 合成原油を精製し、e-ガソリン、e-ディーゼル、e-ジェット燃料などを製造。
- **特徴**:
    - 既存のインフラ（タンクローリー、ガソリンスタンド）や車両（エンジン車）をそのまま活用できる（ドロップイン燃料）。
    - 原料を安定的に確保できれば、大規模な生産が可能。
- **課題**:
    - 製造プロセスで多くのエネルギーを必要とするため、製造コストが高い。
    - 再生可能エネルギーの大量導入が前提となる。

### 3. その他のカーボンニュートラル燃料
- **グリーン水素**:
    - 再生可能エネルギー由来の電力で製造された水素。燃焼時に水しか排出しない究極のクリーンエネルギーですが、貯蔵や輸送に課題があります。
- **グリーンアンモニア**:
    - グリーン水素と窒素を反応させて製造されるアンモニア。燃料として直接利用するほか、水素の輸送・貯蔵媒体（キャリア）としての役割も期待されています。

これらの燃料は、それぞれに利点と課題があり、地域特性や用途に応じて適切なものを選択・組み合わせていく「エネルギーミックス」が、脱炭素社会の実現に向けて重要となります。
