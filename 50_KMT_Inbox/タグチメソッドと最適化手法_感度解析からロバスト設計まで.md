# タグチメソッドと最適化手法：感度解析からロバスト設計まで

[[2025-12-18]]のセミナーメモより、[[タグチメソッド]]（品質工学）を活用した最適化のアプローチについてまとめます。

## 1. 感度レベルの確認とパラメータの絞り込み
*   **[[要因効果図]]（主効果図）**: どの設計パラメータ（要因）が結果に大きく影響しているか（感度が高いか）を可視化する。
*   **[[SN比]]（信号対雑音比）**: テスト結果の信頼性を判断する指標としても使用。SN比が高いほど、ばらつき（ノイズ）に強く安定している。
*   **経験則**: 高橋氏の経験則では、相関係数 **0.85以上** を基準として採用している。

## 2. 多目的最適化とAI活用
*   要因効果図で重要なパラメータを絞り込んだ後、さらに精度を高めるために**ビッグデータ**と**[[機械学習]]**を活用する。
*   絞り込まれたパラメータに対して、どういう組み合わせが最適かを探索させる。

## 3. ロバスト最適化によるコスト削減
*   **感度が低い部分の活用**: 要因効果図で「感度が低い（結果にあまり影響しない）」と分かったパラメータは、厳密に管理する必要がない。
*   **コストダウン**: 感度が低い部分は「ある程度いい加減（公差を緩める、安い材料を使う）」でも品質に影響しないため、ここでコストを下げる。
*   「品質を上げる」だけでなく、「どこで手を抜けるか」を見極めるのもタグチメソッドの重要な役割。

## 4. チームへの適用
*   テスト工程にもタグチメソッドを導入し、試験結果自体の信頼性（SN比）を評価する。
