# コマツのSVM (Standard Variable Margin) 管理とは

**SVM (Standard Variable Margin)** 管理とは、コマツが経営改革の一環として導入した独自の管理会計手法のこと。「**成長とコストの分離**」をコンセプトとし、固定費の増大を防ぎながら利益体質を強化することを目的としている。

## 定義
SVM（標準変動利益）は以下の計算式で定義される。

$$ 	ext{SVM} = 	ext{売上高} - 	ext{変動費 (Standard Variable Cost)} - 	ext{販売直接費} $$

*   **変動費**: 購買部品費、社内加工費など、生産量に比例して変動するコスト。
*   **販売直接費**: 運賃、販売手数料など。

## 導入の背景と目的
2000年代初頭、コマツは売上が伸びても利益が伴わない（固定費が増大してしまう）構造的な課題を抱えていた。これを解決するために導入されたのがSVM管理である。

1.  **固定費の抑制**: 従来の「限界利益」管理では固定費の増加が見えにくかったため、SVMという指標を用いることで、「売上が増えた分だけ固定費も増やして良い」という考え方を断ち切った。
2.  **コスト構造の可視化**: どの製品・事業が真に利益を生み出しているかを明確にし、不採算事業の撤退やリソースの再配分（構造改革）の判断材料とした。
3.  **現場への浸透**: 単なる財務指標にとどまらず、現場レベルでのコスト削減目標としても活用されている。

## 注意点：もう一つのSVM
IT・AI分野の文脈（i-Constructionなど）では、SVMは **Support Vector Machine (サポートベクトルマシン)** という機械学習のアルゴリズムを指す場合がある。建機の自動化や画像診断などで用いられるが、経営管理の文脈でのSVMとは全く別物であるため注意が必要。
